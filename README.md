# Hotel Harmony

A premium hotel management and marketing platform.

## Project Overview

Hotel Harmony is a modern web application built to streamline hotel management, marketing, and operations.

## Technologies Used

This project is built with:

- **Vite** - Fast build tooling
- **TypeScript** - Type-safe JavaScript
- **React** - UI component library
- **shadcn-ui** - Accessible UI components
- **Tailwind CSS** - Utility-first CSS framework

## Getting Started

### Prerequisites

- Node.js & npm installed — [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation & Running Locally

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd hotel-harmony-main

# Step 3: Install dependencies
npm install

# Step 4: Start the development server
npm run dev
```

## Editing the Code

You can edit this project using any IDE of your choice. Open the `src/` folder to find all the application's components, pages, and logic.

## Deployment

Build the production bundle with:

```sh
npm run build
```

The output will be in the `dist/` folder, ready to deploy to any static hosting provider (Vercel, Netlify, etc.).
